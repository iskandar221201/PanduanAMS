### 1. Akses Alamat Domain
Buka browser Anda dan masukkan alamat domain resmi:
* **URL Saat Ini:** `https://app.diredja.my.id`
>*Catatan: Alamat ini dapat berubah sewaktu-waktu sesuai kebijakan. Pastikan Anda mendapatkan informasi domain terbaru jika alamat di atas tidak dapat diakses.*


### 2. Masukkan Kredensial
Gunakan data login yang telah diberikan oleh Administrator:
* **Username**
* **Password**

> **Penting:** Pastikan tidak ada spasi di awal atau di akhir saat memasukkan username dan password untuk menghindari kegagalan login.

### 3. Verifikasi Keamanan (reCAPTCHA)
Jika AMS menampilkan modul **reCAPTCHA** ("I'm not a robot"):
1. Klik pada kotak centang yang tersedia.
2. Jika diminta, selesaikan tantangan visual (seperti memilih gambar lampu lalu lintas, bus, atau zebra cross).
3. Klik **Verify/Verifikasi**.

### 4. Masuk ke Sistem
Klik tombol **Login** atau **Masuk** untuk mengakses Dashboard utama Anda.

## ğŸ›¡ï¸ Tips Keamanan Akun

Untuk menjaga integritas data perusahaan dan keamanan akun pribadi Anda, harap ikuti panduan berikut:

* **Ganti Password Segera:** Lakukan penggantian password saat pertama kali Anda berhasil login. Jangan gunakan password *default* dari administrator dalam jangka waktu lama.
* **Kriteria Password Kuat:** Gunakan minimal 8 karakter dengan kombinasi:
    * Huruf Besar (A-Z)
    * Huruf Kecil (a-z)
    * Angka (0-9)
    * Simbol (Contoh: !, @, #, $, %)
* **Jangan Bagikan Kredensial:** Administrator tidak akan pernah meminta password Anda tanpa alasan yang jelas dan tanpa persetujuan secara tertulis dari pihak perusahaan. Jangan memberikan username atau password kepada siapa pun.
* **Keamanan Perangkat:** 
Selalu lakukan **Logout** setelah selesai menggunakan sistem.
* **Waspada Phishing:** Selalu periksa URL di *address bar* browser Anda sebelum memasukkan data sensitif. Pastikan Anda berada di domain resmi perusahaan.
