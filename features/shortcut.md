## ğŸŒŸ Keunggulan Fitur Shortcut AMS

### 1. Context-Aware Focus
Saat Anda menekan tombol `/`, AMS bukan hanya memfokuskan kursor ke pencarian, tetapi juga akan **otomatis membuka sidebar** jika sebelumnya sedang tertutup (collapsed). Ini memastikan alur kerja tidak terputus.

### 2. Typing Safety
Sistem shortcut AMS cukup cerdas untuk mengetahui kapan Anda sedang mengetik di form. Shortcut navigasi (seperti Alt+P) tidak akan mengganggu saat Anda sedang mengisi nama nasabah atau catatan di kotak input, kecuali Anda sengaja menekan kombinasi tombol tersebut.

### 3. Visual Help Modal
Anda tidak perlu menghafal semua shortcut. Cukup tekan tombol **?** kapan saja, dan AMS akan menampilkan "Cheat Sheet" atau jendela pop-up berisi daftar lengkap shortcut yang tersedia.

## ğŸ› ï¸ Langkah Penggunaan Best Practice

1.  **Gunakan Tangan Kiri:** Letakkan jari di sekitar tombol **Alt** untuk navigasi cepat antar-menu.
2.  **Pencarian Tanpa Klik:** Saat ingin mencari menu "Legalitas" misalnya, tekan `/`, ketik "Lega", lalu tekan **Enter**. Anda berpindah modul dalam kurang dari 2 detik.
3.  **Maksimalkan Ruang Kerja:** Gunakan **Alt + M** untuk menyembunyikan sidebar saat sedang menganalisis tabel data yang lebar agar tampilan layar menjadi lebih luas.

## ğŸ›¡ï¸ Aturan & Mekanisme AMS

*   **Browser Compatibility:** Shortcut ini dioptimalkan untuk browser modern (Chrome, Edge, Firefox).
*   **Conflict Prevention:** AMS menggunakan prefix `Alt` untuk menghindari bentrokan dengan shortcut bawaan browser (seperti Ctrl+P untuk cetak atau Ctrl+S untuk simpan).
*   **Permission Sync:** Menekan shortcut ke menu yang tidak diizinkan untuk Role Anda (misal: Alt+Q untuk staf biasa) tetap akan diarahkan ke halaman "Akses Ditolak" demi keamanan data.

**Q: Apakah shortcut ini bisa diubah sesuai keinginan saya?**
> A: Saat ini daftar shortcut sudah distandardisasi untuk seluruh pengguna AMS agar memudahkan tim IT dalam memberikan dukungan teknis.

**Q: Mengapa shortcut Alt+D tidak berfungsi di laptop saya?**
> A: Beberapa laptop memerlukan penekanan tombol **Fn** tambahan (misal: Fn+Alt+D). Pastikan tidak ada aplikasi lain yang menggunakan shortcut yang sama di laptop Anda.

**Q: Bagaimana jika saya lupa tombol-tombolnya?**
> A: Cukup klik ikon **Keyboard** kecil di bagian Navbar atas (di samping ikon Dark Mode) untuk melihat daftar bantuannya.
